# Automation Exercise Testing Project

This project focuses on manual and automation testing of an e-commerce web application using Selenium WebDriver with Java.

## Key Features
- Designed test scenarios and test cases for core modules
- Covered Login, Registration, Product, Cart, and Checkout functionalities
- Implemented functional and negative test cases
- Automated test scenarios using Selenium WebDriver
- Used XPath locators and synchronization techniques

## Tools & Technologies
- Selenium WebDriver (Java)
- Core Java
- Git & GitHub

## Modules Covered
- Login
- Registration
- Product
- Cart
- Checkout

## Objective
To validate end-to-end user workflows and ensure application quality through effective testing.
